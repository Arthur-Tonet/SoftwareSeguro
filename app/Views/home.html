Software Seguro — Trabalho RA2

HelpIT: chamados de TI com autenticação e segurança aplicada

Sistema web completo para empresas registrarem chamados de suporte, com cadastro por código de empresa, login seguro, CRUD de chamados e proteção contra falhas comuns como SQL Injection, XSS, CSRF e força bruta.

Criar conta Entrar no sistema
Ambiente seguro ativo MVC + PDO + CSRF + hash de senha
  • Cadastro com validação em HTML, JS e PHP
  • Login com bloqueio após 5 falhas
  • Chamados vinculados ao usuário autenticado
  • Auditoria de ações importantes

T2/T3 — Cadastro e Login

Formulários com validação, senha forte, hash, sessão segura, CSRF e controle de tentativas.

🧾

T4 — CRUD de Chamados

Criação, listagem, edição e remoção de chamados usando autorização por usuário.

Camadas do projeto

HTML, CSS e JavaScript integrados com PHP

O HTML fica nas Views PHP, o CSS em public/assets/css/style.css e o JavaScript em public/assets/js/app.js. O PHP processa as regras de negócio no servidor e o JavaScript melhora a experiência do usuário.
